README

This application is built using django framework. It requires the local system to have python installed. 

--INSTALL PYTHON

Install python from -

https://www.python.org/downloads/

REQUIRED PACKAGES

The application also needs some packages to be installed.

Run the following commands in the command prompt.

--INSTALL django

pip install django

--INSTALL django-heroku

pip install django-heroku

--INSTALL requests

pip install requests

--INSTALL pandas

pip install pandas

--INSTALL sklearn

pip install sklearn

--INSTALL nltk

pip install nltk


After succesfully installing the following packages go to the project dirctory in command prompt and run the following command

python manage.py runserver


OPEN localhost:8000 in the browser to use the application.
